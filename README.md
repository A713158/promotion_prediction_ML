# Promotion Prediction ML

## Overview
Machine learning model to predict employee promotions based on historical HR data.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## Model
- Algorithm: Logistic regression, Decision trees, Random forest, and Ensemble methods (Baggging, Boosting, Voting classifier)
- Evaluation metrics: Accuracy, Precision, Recall

## How to Run
1. Install dependencies
2. Run training script
